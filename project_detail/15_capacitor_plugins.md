# Platform Plugins

This project does not use Capacitor or Android plugins.

## Ulauncher Integration

The only platform integration is the Ulauncher Extension API v2 loaded by Ulauncher at runtime.

## Runtime Actions

The extension uses Ulauncher actions to:

- Render result lists.
- Copy selected cheat.sh output to the clipboard.
- Hide the Ulauncher window.
- Open the full cheat.sh URL in the default browser.
